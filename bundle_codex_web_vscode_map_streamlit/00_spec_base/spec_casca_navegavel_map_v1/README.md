# Pacote — Spec da Casca Navegável MAP V1

Este pacote contém a especificação para gerar uma casca navegável em Streamlit usando dados fictícios compatíveis com o banco planejado.

## Arquivo principal

- `SPEC_CASCA_NAVEGAVEL_MAP_V1.md`

## Uso

Use esse Markdown como prompt/briefing para Claude Code, Codex ou outra IA de codificação.

O objetivo é gerar:

- app Streamlit navegável;
- telas principais;
- dados mockados;
- contrato de dados estável;
- componentes visuais;
- navegação para ficha do paciente.

## Importante

Esta spec **não pede implementação de parser real**, Supabase, WhatsApp, login ou deploy.

Ela foca em:

> casca navegável + dados fictícios no formato correto.
